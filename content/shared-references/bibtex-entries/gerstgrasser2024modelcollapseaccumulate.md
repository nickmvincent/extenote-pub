abstract: >-
  Studies whether model collapse is inevitable. Found that collapse occurs when
  replacing real data with synthetic data each generation. However, when
  accumulating synthetic data alongside original real data, models stay stable
  across sizes and modalities. Suggests data accumulation rather than
  replacement as a solution.
