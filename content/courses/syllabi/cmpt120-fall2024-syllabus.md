## Learning Outcomes by Week

### Week 1
Problem solving fundamentals, algorithms, pseudocode, Python basics (comments, print(), headers), IDE usage, file submission

### Week 2
Algorithm design, input/output, variables, string operations, data types, lists, conditional statements (if/elif/else), Boolean expressions, logical operators, software quality, testing, imports

### Week 3
String methods (strip, lower, upper), data type identification, REPL testing, "in" keyword, list creation, for loops, range(), integers, type conversion, nested conditionals, error identification, method chaining

### Week 4
Range with variables, loops for code reduction, integer/float manipulation, variable initialization, accumulator pattern, list length, string-to-integer conversion, arithmetic operations, decimal formatting

### Week 5
File I/O, string splitting, indexing/slicing (lists and strings), numeric comparisons, operator precedence, list operations, nested loops, accumulation patterns, min/max calculations

### Week 7
String immutability, bits/bytes conversion, binary-decimal conversion, hexadecimal, ASCII/Unicode, RGB color storage

### Week 8
Turtle graphics package, function creation with parameters, function calls, color specification, variable scope

### Week 9
Functions returning values, return vs. print distinction, while loops, sentinel-based loops, user input validation

### Week 10
Custom modules, package/module importing, 2D/3D list manipulation, image processing with RGB, Boolean expression returns

### Week 11
List aliases vs. copies, list mutability in functions, recursion basics, recursive functions (factorial, list sum, string reversal, palindrome checking)

### Week 12
Linear search, binary search, recursive binary search

### Week 13
List swapping, selection sort, algorithm analysis, time complexity, Big O notation, merge sort, critical operations identification
