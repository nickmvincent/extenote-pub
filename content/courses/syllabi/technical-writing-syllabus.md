# Technical Writing Syllabus

## Course Description

This course develops technical writing skills essential for computing science students and researchers.

## Weekly Schedule

### Week 1: Introduction & Writing Fundamentals
- Course overview and expectations
- Principles of clear technical writing
- Common pitfalls and how to avoid them

### Week 2: Academic Paper Structure
- IMRaD structure
- Abstracts and introductions
- Literature review techniques

### Week 3: Methods & Results
- Describing methodology clearly
- Presenting quantitative results
- Tables and figures best practices

### Week 4: Discussion & Conclusions
- Interpreting results
- Addressing limitations
- Future work sections

### Week 5: Documentation
- README files
- API documentation
- Code comments

### Week 6: LaTeX & Tools
- LaTeX basics
- Bibliography management
- Collaboration tools

### Week 7: Grant Writing
- Proposal structure
- Specific aims
- Budget justifications

### Week 8: Public Communication
- Blog posts
- Twitter threads
- Press releases
