# CMPT 120 - Introduction to Computing Science and Programming I (Fall 2024)

An introduction to computing science and computer programming, suitable for students with little or no programming background.

## Course Schedule

Classes meet Monday (2:30-3:20) and Thursday (2:30-4:20).

## Assessment

| Component | Weight |
|-----------|--------|
| Coding assignments | 35% |
| Weekly reflections | 5% |
| Midterm and quizzes | 20% |
| Final exam | 40% |

## Weekly Topics

### Weeks 1-5: Foundations
- Week 1: Problem solving, algorithms, Python basics
- Week 2: Variables, data types, conditionals, lists
- Week 3: String methods, loops, error types
- Week 4: Arithmetic, accumulator pattern, string formatting
- Week 5: Indexing, slicing, file I/O

### Week 6: Midterm Review

### Weeks 7-10: Intermediate Concepts
- Week 7: Binary, hexadecimal, ASCII/Unicode, RGB
- Week 8: Turtle graphics, functions, while loops
- Week 9: Return values, nested lists, modules
- Week 10: Custom modules, image processing

### Weeks 11-13: Advanced Topics
- Week 11: List aliasing, recursion introduction
- Week 12: Linear and binary search
- Week 13: Selection sort, merge sort, Big O notation
