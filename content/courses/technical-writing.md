# Technical Writing for Computing Scientists

A course focused on developing technical writing skills for computing science students and researchers.

## Course Overview

This course covers the essential skills needed to communicate effectively in technical and academic contexts.

## Topics

### Academic Writing
- Paper structure and organization
- Literature reviews
- Methodology sections
- Results presentation
- Discussion and conclusions

### Documentation
- README files and project documentation
- API documentation
- User guides and tutorials
- Code comments and docstrings

### Research Communication
- Conference presentations
- Poster sessions
- Grant proposals
- Blog posts and public communication

### Tools & Workflow
- LaTeX and Overleaf
- Markdown and static site generators
- Citation managers (Zotero, Mendeley)
- Version control for writing
