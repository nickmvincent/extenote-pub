# CMPT 120 Fall 2024 - Weekly Readings

Readings are from the Runestone Interactive Python textbook unless otherwise noted.

## Week 1: Introduction to Programming

- "The way of the program"
- "Algorithms"
- "The Python Programming Language"
- "Executing Python in Runestone Textbook"
- "More about programs"
- "Formal and Natural Languages"
- "A Typical First Program"
- "Comments"

**Focus:** Problem-solving fundamentals, algorithm design, Python basics, and program structure.

## Week 2: Variables, Data Types & Conditionals

- Variables, expressions, statements, and data types
- Variable naming conventions and keywords
- Operators, operands, and input handling
- Boolean values and logical operators
- Conditional execution (if/elif/else)
- String concatenation
- List fundamentals and random module usage

**Focus:** Data manipulation, user input, conditional logic, and basic data structures.

## Week 3: Error Types, String Methods & Loops

- Syntax, runtime, and semantic errors
- String methods (strip, lower, upper)
- Membership operators ("in" and "not in")
- For loop iteration
- Range function
- Type conversion functions
- Nested conditionals

**Focus:** Error identification, string manipulation, loop structures, and type handling.

## Week 4: Arithmetic & Accumulation

- Type conversion (review)
- Operators and order of operations
- Variable reassignment and updating
- Accumulator pattern with +=
- String formatting
- List length operations

**Focus:** Numeric operations, reducing code duplication through loops, and the accumulator pattern.

## Week 5: Indexing, Slicing & File Operations

- Index operator for string characters
- Slice operator for strings and lists
- String comparison operations
- List element access and manipulation
- List concatenation and repetition
- Mutable list modification
- File reading and line iteration

**Focus:** Accessing data elements, file I/O, and nested iteration.

## Week 6: Midterm Review

Review of Weeks 1-5 content.

## Week 7: Computer Representation

- String length and immutability
- Loop traversal by index
- Bits and bytes fundamentals
- Binary to decimal conversion
- Hexadecimal representation
- ASCII and Unicode standards
- RGB color encoding

**Focus:** Understanding computer data storage and representation systems.

## Week 8: Turtle Graphics & Functions

- Turtle module basics and drawing
- Multiple turtle instances
- Iteration with turtle graphics
- Function creation with parameters
- Function scope and variable locality
- While loops and sentinel validation
- RGB color specification

**Focus:** Visual programming, function definition, and iteration concepts.

## Week 9: Functions & Advanced Iteration

- Functions returning values
- Function composition and execution flow
- Nested lists and list mutability
- Python modules and importing
- Image processing theory
- Two-dimensional data structures

**Focus:** Return values versus print, nested data structures, and module usage.

## Week 10: Modules & Image Processing

- Creating custom modules
- Math module usage
- Image processing with 2D and 3D lists
- Pixel manipulation using RGB values
- The cmpt120images module

**Focus:** Code organization through modules and practical image data manipulation.

## Week 11: List Operations & Recursion Introduction

- Object references and aliasing
- List parameters in functions
- Functions producing lists
- Append versus concatenate operations
- Three laws of recursion
- Recursive visualization with turtle
- Simple recursive functions

**Focus:** Understanding reference semantics and introducing recursive problem-solving.

## Week 12: Advanced Recursion & Search Algorithms

- Recursive list processing
- Integer-to-string conversion recursively
- Linear search implementations
- Binary search algorithm
- Recursive binary search

**Focus:** Recursive patterns and fundamental search techniques.

## Week 13: Sorting & Algorithm Analysis

- Selection sort implementation
- Merge sort algorithm
- Algorithm analysis fundamentals
- Big O notation
- Time complexity calculation
- Comparing algorithm efficiency
- Critical operations identification

**Focus:** Sorting strategies and quantifying algorithm performance.
